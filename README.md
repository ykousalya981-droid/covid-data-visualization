# COVID-19 Data Analysis Project

## 📌 Overview
This project analyzes COVID-19 testing data using Python.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Features
- Top 10 countries by testing
- Pie chart distribution
- Scatter plot (tests vs positive)
- India vs USA comparison

## 📈 Insights
- Higher testing leads to higher detection of cases
- Strong relationship between tests and positive cases

## 🚀 How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python covid_analysis.py
